## Key UI Features

### 1. Smart Visibility
- Setup guide only shown when not connected
- Automatically hidden after successful connection
- Clean interface without clutter

### 2. Progressive Disclosure
- Collapsible setup guide
- Expand/collapse with chevron icons
- Numbered steps for easy navigation

### 3. Copy-Paste Friendly
- Pre-generated URLs
- One-click copy buttons
- Code blocks with proper formatting

### 5. Context-Aware
- Shows current app URL automatically

- Adapts to deployment environment

### 6. User Guidance
- Step-by-step instructions
- What happens next preview
- Warning messages for critical info
- Success indicators

### 7. Documentation Links
- Quick reference to detailed docs
- Multiple documentation options
- In-app and external resources

## Color Scheme

- **Blue** (`blue-50`, `blue-600`): Info, setup, primary actions
- **Green** (`green-50`, `green-600`): Success, connected, active
- **Yellow** (`yellow-50`, `yellow-200`): Highlight important URLs
- **Red** (`red-50`, `red-900`): Warnings, critical info
- **Gray** (`gray-50`, `gray-900`): Code blocks, neutral content

---

## Responsive Behavior

The interface adapts to different screen sizes:

- **Desktop**: Full width with optimal spacing
- **Tablet**: Stacked layout with maintained readability
- **Mobile**: Scrollable setup guide with touch-friendly buttons

---

## Interaction States

### Buttons
- **Default**: Blue with white text
- **Hover**: Darker blue
- **Disabled**: Gray with reduced opacity
- **Loading**: Spinner animation

### Collapsible Section
- **Collapsed**: Down chevron, hidden content
- **Expanded**: Up chevron, visible content
- **Transition**: Smooth animation

### Copy Button
- **Default**: Blue link with icon
- **Click**: Alert confirmation
- **Post-click**: "Copied to clipboard!" message

---

This interface provides a seamless setup experience directly within the application, eliminating the need to reference external documentation during initial configuration.
