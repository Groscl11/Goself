Strategy
Placement Priority & Conversion Impact
Where to implement first for maximum ROI.
Surface Intent Level Primary Goal Data Needed Priority
🛒 Cart Page Widget Highest Nudge to add more items Cart total, threshold Build First
📱 Sticky Mobile Bar High Always-on awareness Cart total, threshold Build First
🏷 Product Page Strip Medium-High Convert consideration to add Cart total, threshold Phase 2
🏠 Homepage Hero Medium Brand trust + awareness Campaign + brand data Phase 2
📂 Collection Banner Low-Medium Broad reach, browsing nudge Campaign data only Phase 3

⚡ How to power these surfaces from your existing data
Cart/Product progress bar: Shopify cart API gives current total. Your campaign_rules table has the threshold. Simple subtraction = "₹X away".

Brand logos + offer names: campaign_reward_pools → rewards → brands. Already in DB. Pull once per page load and cache.
Sticky bar on mobile: Shopify theme section or app block. Reads same cart total. No backend call needed until customer clicks "See Rewards".

Email template: Triggered from shopify-webhook → communication_logs. Already queued — just need HTML template wired to it.
