Reward Widget Test Page

Test your widgets before deploying to Shopify

Instructions

Follow these steps to test your widget:

  1. Get your Widget ID from the Shopify Widgets page
  2. Replace YOUR_WIDGET_ID_HERE in the code below with your actual widget ID
  3. Add the script tag to this page (see examples below)
  4. Reload the page to see your widget

Test: Floating Widget

This widget appears as a floating button in the bottom-right corner.

<script src="/widget.js" data-widget-id="YOUR_WIDGET_ID_HERE"></script>
Expected Result: A floating reward card should appear in the bottom-right corner of the page.

Test: Thank You Card

This widget appears inline in the page content.

Thank You Card widget will appear here
<div id="reward-widget-YOUR_WIDGET_ID_HERE"></div> <script src="/widget.js" data-widget-id="YOUR_WIDGET_ID_HERE"></script>

Test: Product Banner

This widget appears as a promotional banner.

Product Banner widget will appear here

Test: Announcement Bar

This widget appears at the top of the page.

Note: The announcement bar widget automatically inserts itself at the top of the page.

Troubleshooting

  • Check browser console (F12) for any error messages
  • Verify the widget ID is correct and the widget is enabled
  • Make sure the widget.js file is loading (check Network tab in browser dev tools)
  • Ensure your browser allows JavaScript execution

Next Steps

Once the widget displays correctly here:

  1. Copy the embed code from the Shopify Widgets page
  2. Follow the installation instructions for your widget type
  3. Add the code to your Shopify theme
  4. Test on your Shopify store
